# Core Data Structures and Algorithms
This module contains implementations of fundamental data structures and algorithms used in computer science and software engineering.

All implementaions are based on foundational concepts and include unit tests.

---

## Module Structure

    .
    ├── algorithms                  # Algorithm implemenations such as BFS, DFS, Dijkstra, etc.
    ├── arrays                      # Array-related data structures and utiliites
    ├── graphs                      # Graph data structures and helper methods
    └── heap                        # Heap implementations

---

## Data Strutures

Implemented data structures include:

- Arrays (Dynamic Array)
- Graphs (Adjaceny List)
- Heap (Prioty Queue)
